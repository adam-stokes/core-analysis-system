Welcome, sleekFMK is an attempt at making a web framework self contained and easy to develop against. Each sleekFMK application is run in its own environment which allows stability and consistency across systems/platforms. The codebase is Python and has been tested on versions 2.6.x and 2.7.x. Some of the technologies behind sleekFMK include tornado web server and mongodb. The code is hosted at bitbucket.org and can be followed through their means.

What's supported right out of the box?

  • Document oriented database
  • Natively compiled templates
  • Themes
  • Encryption
  • Authentication through Twitter, Google, Facebook
  • Asynchronous requests
  • Extensible through whatever means, we dont limit you to plugin hooks :)
  • Unittests
  • Standalone or WSGI
  • Easily scalable

Download our installer

If you have wget or curl you can grab our installer to get started with the framework.

wget https://bitbucket.org/battlemidget/ajscg-tools/raw/tip/ajscg-python/sleekinstaller/sleekinstaller
curl -C - -O https://bitbucket.org/battlemidget/ajscg-tools/raw/tip/ajscg-python/sleekinstaller/sleekinstaller

Next, make it executable or pass it through python
# python sleekinstaller full project-name
